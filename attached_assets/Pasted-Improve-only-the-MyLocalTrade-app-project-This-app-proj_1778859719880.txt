Improve only the MyLocalTrade app project.

This app project is separate from the public landing page mylocaltrade.co.uk. Do not edit the standalone landing page project.

Goal:
Improve the app into a trusted local services marketplace where customers can find verified traders, request quotes, manage enquiries, and where traders can manage leads, subscriptions and verification.

Please inspect the existing app first. If something is already implemented, do not rebuild it. Only improve, fix, or complete what is missing.

Main app improvements:

1. Customer home screen/dashboard
Add or improve:
- Search for local trades
- Popular categories
- Verified traders
- Emergency services
- Energy upgrades
- Property maintenance
- Recent enquiries
- Saved traders
- Clear CTA: Request a quote

2. Trader dashboard
Add or improve:
- New leads
- Enquiry status
- Profile completion score
- Verification status
- Subscription status
- Upgrade prompts
- Gallery/profile management
- Service areas
- Specialist badges

3. Verified trader system
Add or improve trader verification statuses:
- Not verified
- Pending verification
- Verified trader
- Premium verified trader

Verification fields should include:
- Business name
- Company number if applicable
- Public liability insurance
- Trade qualifications
- Years of experience
- Service areas
- Insurance expiry date
- Admin approval status

Only admin should approve verification.

- Business description

- Photos/gallery
- Verification badges
- Insurance and qualification indicators
- Service areas
- Contact/request quote button

- Energy upgrade specialist option
- Property maintenance specialist option
